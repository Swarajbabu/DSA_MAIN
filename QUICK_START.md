# 🚀 Quick Start Guide

Get started with DSA_MAIN in just a few minutes!

## ⚡ Quick Setup

### 1. Clone the Repository
```bash
git clone https://github.com/Swarajbabu/DSA_MAIN.git
cd DSA_MAIN
```

### 2. Choose Your Compiler

**Windows (G++):**
```bash
# Download MinGW or use: choco install mingw
g++ --version
```

**Mac (Clang):**
```bash
# Already installed with Xcode
clang++ --version
```

**Linux (G++):**
```bash
sudo apt-get install build-essential
g++ --version
```

## 📝 Compile & Run Examples

### Arrays
```bash
cd "1. ARRAYS"
g++ -o arrays_output arrays_.c++
./arrays_output
```

### Linked Lists
```bash
cd "2. LinkedList"
g++ -o sll_output SLL.c++
./sll_output
```

### Stacks
```bash
cd "3. Stacks"
g++ -o stack_output stacks.c++
./stack_output
```

### Recursion
```bash
cd Recursion
g++ -o recursion_output Recursion.c++
./recursion_output
```

## 🆘 Troubleshooting

**"Command not found: g++"**
- Install a C++ compiler (see Setup section)
- Add compiler to PATH

**"Compilation error"**
- Check your C++ standard: `g++ -std=c++17 file.c++`
- Review the code syntax

**"Undefined reference"**
- Check for missing function implementations
- Ensure all includes are present
